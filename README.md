# Projeto de Rede Corporativa - Cisco Packet Tracer

## Descrição

Este projeto consiste na simulação de uma rede corporativa desenvolvida no Cisco Packet Tracer como atividade prática de conclusão de curso.

A proposta foi criar uma infraestrutura segmentada por setores, utilizando VLANs e serviços de rede essenciais para o funcionamento de um ambiente organizacional.

---

## Estrutura da Rede

A rede foi dividida em três setores:

- VLAN 10 – Administrativo (192.168.10.0/24)
- VLAN 20 – Laboratório (192.168.20.0/24)
- VLAN 30 – Alunos (192.168.30.0/24)

A comunicação entre switches foi realizada utilizando portas configuradas como trunk (802.1Q).

---

## Serviços Implementados

- Servidor DHCP para distribuição automática de IP
- Servidor DNS
- Servidor WEB
- Rede Wi-Fi integrada à infraestrutura
- Segmentação lógica por VLAN
- Organização da topologia por setores

---

## Objetivos do Projeto

- Aplicar conceitos de VLAN e segmentação de rede
- Configurar trunk entre switches
- Implementar serviços básicos de infraestrutura
- Simular um ambiente corporativo funcional
